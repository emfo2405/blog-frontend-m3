## Blog frontend
För den här uppgiften skapade jag en blogg som använder ett API publicerat på Render. All data som presenteras och som kan redigeras på
sidan läses in från API:et och använder CRUD-funktionalitet. På sidan kan man läsa blogginlägg, komma åt startsidan och inloggningssidan. 
För en inloggad användare finns även en adminsida där man kan skapa nya inlägg samt möjlighet att, från bloggsidan, redigera
eller radera befintliga inlägg.
